# how to run this project?

## 1. open your terminal & redirect to 'vue-app'

    cd ./vue-app

## 2. install packages

    npm i

## 3. run the project

    npm run dev

#### note:

    1. running 'npm run format' will trigger prettier
    2. i didn't create back-end app, I just use public api & fetch in the project
    3. token.js is located at utils folder inside /src
